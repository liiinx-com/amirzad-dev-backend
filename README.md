# Personal Portfolio Backend

## Requirements

1. Node > 16.X

## RUN

1. Clone the repository
2. `cd ./amirzad-dev-backend`
3. `cp .env.sample .env`
4. fill the values in the ./.env file
5. `npm install`
6. `npm start`
7. visit http://localhost:3000
